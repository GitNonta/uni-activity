# 🎨 Show Custom Error Pages - Quick Start

## ⚡ Super Quick (Choose One)

### 1️⃣ Double-Click This File:
```
show-custom-errors.bat
```

### 2️⃣ Or Run This Command:
```powershell
.\toggle-debug.ps1 off
```

### 3️⃣ Or Edit .env Manually:
Change `APP_DEBUG=true` to `APP_DEBUG=false`

Then run: `php artisan config:clear`

---

## ✅ Test Your Error Pages

Visit: **http://localhost:8000/test-errors**

---

## 🔄 Switch Back to Debug Mode

Double-click: `show-debug-mode.bat`

Or run: `.\toggle-debug.ps1 on`
